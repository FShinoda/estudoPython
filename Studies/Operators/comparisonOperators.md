# Comparision Operator

The comparision operators are used to compare variables to values or variables to variables.

| Operator | Name | Example |
|----------|------|---------|
| '==' | 'Equal' | 'x == y' |
| '!=' | 'Not equal' | 'x != y' |
| '>' | 'Greater than' | 'x > y' |
| '<' | 'Less than' | 'X < y' |
| '>=' | 'Greater than or equal to' | 'x >= y' |
| '<=' | 'Less than or equal to' | 'x <= y' |
